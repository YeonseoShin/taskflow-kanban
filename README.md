# TaskFlow Kanban Board

외부 라이브러리 없이 HTML5 Drag & Drop API와 Vanilla JS만으로 구현한 칸반 보드입니다.

## 기술 스택
- Vanilla JS (HTML5 Drag & Drop API)
- HTML / CSS

## 주요 기능
- 컬럼 간 카드 드래그 앤 드롭
- 카드 추가 / 완료 처리
- 컬럼별 카드 수 실시간 업데이트

## 파일 구조
```
taskflow-kanban/
├── index.html    # 보드 구조
├── style.css     # 스타일
└── kanban.js     # 드래그 앤 드롭 로직
```
